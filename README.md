# 部位汇总（按日）— v5-full

- 内置你的《物料清单完整.xlsx》（sheet=映射），即使不上传映射表也可直接使用。
- 支持上传“净重台账 / 生产日报”，用于计算**产成率%**（成品产量/毛鸡净重）。
- 自动输出：按日·部位汇总、未识别Top20（按金额）、以及可下载的 Excel。

## 快速开始
```bash
pip install -r requirements.txt
python app_gradio_full.py
# 浏览器打开 http://127.0.0.1:7887
```

**字段自动识别**  
- 日期列候选：日期/交鸡日期/记账日期/单据日期/出库日期/…  
- 金额列候选：含税金额/金额/本位币金额/产值/…  
- 数量列候选：数量/净重/出库数量/…  
- 名称列候选：物料描述/品名/名称/…  

## 文件说明
- `app_gradio_full.py`：Gradio 应用（已启用“映射表回退”）。
- `物料清单完整.xlsx`：你的映射表（默认使用【映射】sheet 两列：关键词、部位）。
- `示例_主数据模板.xlsx`、`示例_净重台账.xlsx`：示例数据，方便校验流程。
- `输出/`：运行后生成的 Excel 将保存到此目录中（程序启动会自动创建）。

## 提示
出现“未识别Top20”时，在【映射】sheet 增加一行（关键词、部位），**越具体越靠上**，再运行一次即可。
